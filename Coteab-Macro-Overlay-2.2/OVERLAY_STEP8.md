# Native Overlay Step 8

The macro's **Other Features → Native In-Game Overlay** panel now controls the native overlay directly.

Working controls:
- Enable/disable overlay
- Expanded or compact layout
- Top-right, bottom-right, or custom position
- Background-only transparency
- Midnight, Crimson, Emerald, Solar, Frost, and AMOLED themes
- Custom accent color
- Font selection
- Typeable minimum aura rarity up to 10,000,000,000
- Individual visibility switches for status, biome, aura, rarity, merchant, merchant elapsed time, local time/day-night, and session
- Width and height controls
- Snap and reset buttons

The existing biome, aura, and merchant detection integrations are unchanged.
