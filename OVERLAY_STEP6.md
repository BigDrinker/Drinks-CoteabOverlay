# Native Overlay Step 6

Adds live Last Aura and Aura Rarity to the proven native overlay.

The values come from the macro's existing aura detection state:
- `overlay_last_aura_found`
- `overlay_last_aura_rarity`

The existing typeable `overlay_minimum_rarity` setting controls which detected auras are allowed to replace the overlay's last-aura display.
