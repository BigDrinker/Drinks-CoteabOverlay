Coteab Native Overlay 2.0

This build removes the legacy WebView/hybrid overlay settings UI.
All native overlay preferences now live in overlay_settings.json.

Run:
  py -3.12 -m pip install -r requirements.txt
  py -3.12 main.py

Use Other Features > Overlay 2.0.
The overlay itself can be moved by its header and resized from every edge/corner.
