Overlay 2.1 additions

- Profiles: Minimal, AFK, Streaming, Full, and Custom
- User-controlled row ordering with up/down controls
- Compact and expanded layouts
- Optional biome, aura, and merchant notifications
- Session biome-change and merchant counters
- Export settings to overlay_profile_export.json
- Import settings from overlay_profile_export.json

Notes:
- Counters reset whenever the overlay process restarts.
- Export/import uses a JSON file beside main.py so it remains easy to share and inspect.
- Existing live biome, aura, merchant, status, and session integrations are preserved.
