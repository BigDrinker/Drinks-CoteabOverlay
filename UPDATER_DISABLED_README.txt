Coteab Macro v2.2 Overlay - Custom Build

The automatic updater is intentionally disabled in this build.
Reason: the upstream updater downloads and restarts into the latest official Noteab/Coteab release, which can overwrite this custom overlay version and make the new features disappear.

To update this custom build, download a newer custom package and rebuild it manually.

The included build_coteab_exe.bat also bundles both AutoIt DLL files automatically.
